#### Question-1️⃣ What is the difference between var, let, and const?

#### Solution_1: All used for declare variable. But they have different scope, redeclaration, reassignment and hoisting.

# var: It is function scoped, can be redeclared/reassignment, hoisting is undefined.

# let: It is Block scoped, can't be redeclared/reassigned, hoisting but in temporal dead zone.

# const: It also block scoped, can't be redeclared/reassigned, and must be initialize when declared.

#### Question-2️⃣ What is the spread operator (...)?

#### Solution_2: Spread operator is used to expand arrays, objects into individual elements.

#### Question-3️⃣ What is the difference between map(), filter(), and forEach()?

#### Solution_3:

# map() transforms each element an array and returns a new array. Does't change the original array.

# filter() select elements from an array based on a condition and return new array.

# forEach() iterate an array but does't return a new array.

#### Question-4️⃣ What is an arrow function?

#### Solution_4: An arrow function is a shorter way to write functions in JS using the => syntax.

# Example:

# normal function- function add(a, b) { return a + b; }

# an arrow function- const add = (a, b) => a + b;

#### Question-5️⃣ What are template literals?

#### Solution_5: Template literals are a way to create strings in JS using backticks sign `` that allow variable interpolation and multi-line strings. ${} is used to insert variable or expression inside a string.
